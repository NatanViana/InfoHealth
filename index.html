Política de Privacidade - InfoHealth

O App utiliza um serviço de banco de dados do Google chamado Firebase Database.

Ele coleta as informações do usuário para guarda-las em ambinte seguro com o intuito de melhorar a perfomance do app.

O App utiliza o GPS para localizar o usuário e decidir se a localização é a mais apropriada para o envio de notificações.

O App utiliza também a API de localização para converter endereços em latitude e longitude para o comparar com a localização do usuário

Exclusão da Conta - InfoHealth
